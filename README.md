## Cómo ejecutar el juego

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/PixelInvaders.git
   ```
2. Navega a la carpeta del proyecto:
	```bash
	cd PixelInvaders
	```
3. Ejecuta la Build del juego en la carpeta "Game Build".
